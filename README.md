# Plataforma SaaS de Gestión de Inventario
Sistema back end para la gestion de inventario basado en plantillas con autenticacion cont tokens JWT.

## Tecnologías
- Python
- Django
- Django REST Framework
- PostgreSQL
- JWT Authentication

## Infraestructura
- Git

## Funcionalidades
- Autenticación JWT y gestión de cuentas
- CRUD de categorías, plantillas y productos
- Rutas de reportes sobre antradas y salidas de inventario
- Flujo inicial de onboarding
- Integración frontend–backend vía API REST

## Arquitectura
Proyecto dividido en dos repositorios:

- inventory-saas → Backend (Django REST Framework)
- inventory_saas_client → Frontend (React / Next.js)
